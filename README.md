# ThreeJS-Solar-Farm

Simple 3d animation of a solar panel farm on a day-night cycle.

## Features

- Day-night cycle
- Solar panels rotate according to the position of the sun
- Added shadows
- Orbit Controlls

### How to open

- Install: `npm install`
- Run: `npm start`

### Stack

- JavaScript
- React
- Three.js
- React-three-fiber
- HTML
- CSS
